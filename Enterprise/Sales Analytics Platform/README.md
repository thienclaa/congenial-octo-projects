# Sales Analytics Platform

## Overview

Enterprise Power BI reporting solution developed for Long Châu Pharmacy E-Commerce to monitor sales performance, customer behavior, promotions, and payment analytics across approximately 100K+ daily sales transactions.

## Business Impact

- Supported 100K+ daily sales transactions
- Built one of the company's primary executive sales dashboards
- Served 100+ business users
- Increased call center revenue by 15%
- Reduced semantic model refresh from 90 to 30 minutes
- Reduced semantic model storage from 15 GB to 5 GB

## Technology

- SQL Server
- Azure Synapse Analytics
- Databricks
- Power BI
- DAX
- Python
